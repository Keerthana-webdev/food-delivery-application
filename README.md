# 🍔 Food Delivery MERN Appliation

A modern food delivery web application built using the MERN stack.

## 🚀 Features
- Browse restaurants
- View food items
- Add items to cart
- Cart management
- Responsive UI
- Category filters
- Search functionality

## 🛠 Tech Stack
Frontend:
- React.js
- Tailwind CSS

Backend:
- Node.js
- Express.js

Database:
- MongoDB

## ⚡ Installation

Clone the repository

git clone https://github.com/Keerthana-webdev/food-delivery-application.git

Install dependencies

cd client  
npm install  

Run the project
npm start

cd server  
npm install  

Run the project
node server.js
